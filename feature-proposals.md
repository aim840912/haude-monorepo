## 1. 互動式數據儀表板

### 概述

將現有的靜態統計卡片升級為動態、互動式的數據視覺化儀表板，讓管理者一眼掌握營運全貌。

### 功能特色

#### 1.1 營收趨勢圖

- **日視圖**：顯示當日每小時營收變化
- **週視圖**：顯示本週每日營收，與上週同期比較
- **月視圖**：顯示本月每日營收曲線，標記異常值
- **互動功能**：滑鼠懸停顯示詳細數據，支援區間選取放大

#### 1.2 熱銷產品排行榜

- 即時更新的 Top 10 熱銷產品
- 顯示銷量、營收、庫存狀態
- 點擊可快速跳轉至產品編輯頁
- 動態排名動畫效果

#### 1.3 訂單狀態分佈

- 圓餅圖展示各狀態訂單佔比
- 待處理訂單醒目提示
- 點擊可篩選對應狀態訂單列表

#### 1.4 會員增長曲線

- 新註冊會員趨勢
- 會員活躍度分析
- VIP 會員佔比

#### 1.5 KPI 數字動畫

- 今日營收、訂單數、新會員等關鍵指標
- 數字滾動動畫效果
- 與昨日/上週同期比較，顯示成長率

### 設計亮點

| 特點         | 說明                                 |
| ------------ | ------------------------------------ |
| **即時更新** | 支援自動刷新（可設定間隔）或手動刷新 |
| **響應式**   | 桌面版完整圖表，手機版精簡卡片       |
| **深色模式** | 完整支援深色主題                     |
| **載入動畫** | 優雅的骨架屏載入效果                 |

### 技術選型

- **圖表庫**：Recharts（React 生態系最佳選擇）或 Chart.js
- **動畫**：Framer Motion 數字滾動效果
- **數據快取**：React Query 優化重複請求

### 客戶價值

- 老闆/管理者最常看的頁面，視覺衝擊力直接影響對系統的評價
- 展示前端數據視覺化能力
- 為決策提供數據支持

### 預估工時

2-3 天（含 API 端點調整）

## 2. 互動式地圖整合

### 概述

將農場據點資訊與互動式地圖結合，提供沉浸式的地理位置瀏覽體驗。

### 功能特色

#### 2.1 據點地圖視圖

- 地圖上標記所有農場據點（茶園、市集點）
- 自訂標記圖示（區分不同類型據點）
- 點擊標記展開資訊卡片
- 支援地圖縮放和拖曳

#### 2.2 據點資訊卡片

- 據點名稱、照片輪播
- 營業時間、聯絡方式
- 特色產品/體驗
- 「導航至此」按鈕（開啟 Google Maps）

#### 2.3 路線規劃

- 從使用者當前位置規劃路線
- 顯示距離和預估時間
- 支援多種交通方式（開車/大眾運輸）

#### 2.4 附近景點推薦

- 整合周邊觀光景點
- 一日遊行程建議
- 提升農場的觀光價值

#### 2.5 行動端優化

- 「附近的農場」功能（基於 GPS 定位）
- 距離排序
- 地圖/列表視圖切換

### 設計亮點

| 特點               | 說明                           |
| ------------------ | ------------------------------ |
| **客製化地圖風格** | 配合品牌色調的地圖配色         |
| **流暢動畫**       | 標記出現、資訊卡彈出的過渡效果 |
| **離線快取**       | 基本據點資訊可離線查看         |
| **分享功能**       | 分享特定據點連結給朋友         |

### 技術選型

- **地圖服務**：Google Maps API 或 Mapbox（更好客製化）
- **定位**：瀏覽器 Geolocation API
- **資料來源**：現有 Location 模型（已有 lat/lng 欄位）

### 客戶價值

- 視覺震撼力強，提升品牌專業形象
- 符合農場/觀光產業的核心需求
- 展示第三方 API 整合能力

### 預估工時

2-3 天（含地圖樣式客製化）

## 3. 銷售報表系統

### 概述

提供完整的銷售數據分析與報表導出功能，支援企業決策。

### 功能特色

#### 3.1 銷售日報

- 當日訂單明細
- 銷售額、訂單數、平均客單價
- 付款方式分佈
- 熱銷商品當日排名

#### 3.2 銷售週報/月報

- 營收趨勢對比（本週 vs 上週，本月 vs 上月）
- 成長率計算與視覺化
- 異常值標記與分析
- 季節性趨勢洞察

#### 3.3 產品銷售分析

- 各產品銷量、營收貢獻度
- 產品組合分析（哪些產品常一起購買）
- 庫存周轉率
- 滯銷品預警

#### 3.4 會員消費分析

- 會員消費頻率分佈
- 高價值會員識別（RFM 分析）
- 會員流失預警
- 復購率統計

#### 3.5 報表導出

- Excel 格式（.xlsx）
- PDF 格式（含圖表）
- 自訂日期範圍
- 排程自動發送（每日/週/月）

### 設計亮點

| 特點           | 說明                           |
| -------------- | ------------------------------ |
| **多維度篩選** | 時間範圍、產品分類、會員等級等 |
| **同比環比**   | 自動計算成長率並標色           |
| **視覺化摘要** | 關鍵數據優先展示               |
| **行動友好**   | 手機也能查看報表               |

### 技術選型

- **報表生成**：後端計算 + 前端渲染
- **Excel 導出**：xlsx 或 exceljs
- **PDF 導出**：puppeteer 或 html-pdf
- **排程任務**：Node.js cron 或外部排程服務

### 客戶價值

- 老闆最關心的功能，直接影響商業決策
- 展示後端數據處理與聚合能力
- B2B 客戶必備需求

### 預估工時

3-4 天（含多種報表格式）

## 4. 智慧通知系統

### 概述

建立自動化的通知機制，減少人工監控成本，確保關鍵事件即時處理。

### 功能特色

#### 4.1 庫存預警

- 設定各產品的庫存警戒線
- 低於警戒線自動發送通知
- 預估售罄時間（基於銷售速度）
- 建議補貨數量

#### 4.2 訂單通知

- 新訂單即時提醒（站內 + Email）
- 大額訂單特別標記
- 訂單狀態變更通知
- 待處理訂單提醒（超過 X 小時未處理）

#### 4.3 會員通知

- 新會員註冊通知
- VIP 會員下單通知
- 會員生日提醒
- 長期未活躍會員提醒

#### 4.4 系統通知

- 支付異常警報
- 系統錯誤日誌摘要
- 每日營運摘要郵件

#### 4.5 通知中心

- 站內通知列表（未讀/已讀狀態）
- 通知偏好設定（選擇接收哪些通知）
- 通知歷史查詢
- 批量標記已讀

### 設計亮點

| 特點       | 說明                               |
| ---------- | ---------------------------------- |
| **多管道** | 站內通知 + Email + 未來可擴展 LINE |
| **可配置** | 每種通知可單獨開關                 |
| **即時性** | 關鍵通知即時推送                   |
| **優先級** | 按重要程度分類展示                 |

### 技術選型

- **站內通知**：資料庫儲存 + 輪詢或 WebSocket
- **Email**：現有 EmailService + 模板系統
- **排程檢查**：定時任務檢查庫存等條件

### 客戶價值

- 減少人工監控成本
- 避免缺貨、訂單延誤等問題
- 展示系統自動化能力

### 預估工時

2-3 天（核心功能）

## 5. 會員等級系統

### 概述

完整實作會員等級與積分系統，提升客戶忠誠度與回購率。

### 功能特色

#### 5.1 會員等級

| 等級         | 條件               | 權益                          |
| ------------ | ------------------ | ----------------------------- |
| **普通會員** | 註冊即獲得         | 基礎服務                      |
| **銅卡會員** | 累積消費 NT$3,000  | 95 折、生日禮                 |
| **銀卡會員** | 累積消費 NT$10,000 | 9 折、優先預約、生日禮        |
| **金卡會員** | 累積消費 NT$30,000 | 85 折、專屬客服、生日禮、免運 |

#### 5.2 積分系統

- **獲取規則**：消費 NT$1 = 1 點
- **額外積分**：生日月雙倍積分、指定活動加倍
- **積分兌換**：可兌換商品、折扣券、體驗活動
- **積分有效期**：獲得後 1 年內有效

#### 5.3 會員專屬優惠

- 等級專屬折扣自動套用
- 生日當月專屬優惠券
- 會員專屬商品（限定款）
- 新品優先購買權

#### 5.4 等級進度追蹤

- 當前等級與下一等級差距
- 進度條視覺化
- 近期升級可能性提示

#### 5.5 會員中心

- 等級狀態展示
- 積分餘額與歷史
- 可兌換商品列表
- 專屬優惠券領取

### 設計亮點

| 特點           | 說明                   |
| -------------- | ---------------------- |
| **自動升降級** | 每月自動計算並調整等級 |
| **視覺化進度** | 清晰展示升級進度       |
| **多管道通知** | 升級時發送祝賀通知     |
| **管理員控制** | 後台可手動調整會員等級 |

### 技術選型

- **資料模型**：擴展現有 User 模型，新增積分相關表
- **計算邏輯**：定時任務計算等級變化
- **折扣套用**：整合現有 DiscountsService

### 客戶價值

- 提升客戶忠誠度和回購率
- 電商標配功能，展示完整性
- 展示複雜業務邏輯處理能力

### 預估工時

4-5 天（完整功能）

## 6. AI 智慧推薦系統

### 概述

利用機器學習或規則引擎為用戶提供個性化的產品推薦，提升轉換率。

### 功能特色

#### 6.1 「猜你喜歡」推薦

- 基於用戶瀏覽歷史
- 基於購買記錄
- 基於相似用戶偏好
- 展示於首頁和產品詳情頁

#### 6.2 「購買此商品的人也買了」

- 協同過濾算法
- 購物車頁面交叉銷售
- 訂單完成後推薦

#### 6.3 個性化首頁

- 根據用戶偏好調整首頁產品排序
- 新用戶展示熱銷品
- 回訪用戶展示相關品

#### 6.4 搜尋結果優化

- 個性化排序
- 相關產品補充
- 「您可能還想找」建議

### 推薦演算法層級

| 層級     | 方法                           | 複雜度 | 效果 |
| -------- | ------------------------------ | ------ | ---- |
| **基礎** | 規則推薦（同分類、標籤匹配）   | 低     | 中   |
| **進階** | 協同過濾（基於用戶行為相似度） | 中     | 高   |
| **AI**   | Embedding 向量相似度（OpenAI） | 高     | 最佳 |

### 設計亮點

| 特點           | 說明                         |
| -------------- | ---------------------------- |
| **冷啟動處理** | 新用戶使用熱門推薦           |
| **實時更新**   | 用戶行為即時影響推薦         |
| **多場景**     | 首頁、詳情頁、購物車、結帳後 |
| **A/B 測試**   | 支援推薦策略對比測試         |

### 技術選型

- **基礎版**：SQL 查詢 + 標籤匹配
- **進階版**：Redis 快取 + 定時計算相似度矩陣
- **AI 版**：OpenAI Embedding API + 向量資料庫（Pinecone/Supabase pgvector）

### 客戶價值

- AI 是當前最熱門的技術話題
- 直接提升轉換率和客單價
- 展示機器學習整合能力

### 預估工時

- 基礎版：2-3 天
- 進階版：4-5 天
- AI 版：5-7 天

## 7. LINE 官方帳號整合

### 概述

整合 LINE 平台，打通台灣最主流的社交通訊管道。

### 功能特色

#### 7.1 LINE 登入

- 一鍵 LINE 授權登入
- 自動關聯現有帳號（相同 Email）
- 取得 LINE 頭像和暱稱

#### 7.2 訂單狀態推播

- 訂單成立確認
- 出貨通知（含物流追蹤連結）
- 訂單完成提醒（邀請評價）
- 異常狀態通知

#### 7.3 LINE Notify 庫存預警

- 管理員綁定 LINE Notify
- 庫存低於警戒線推播
- 每日營運摘要推播

#### 7.4 行銷訊息推送

- 新品上市通知
- 促銷活動推播
- 會員專屬優惠
- 推送頻率限制（避免擾民）

#### 7.5 LINE LIFF 小程式（進階）

- 在 LINE 內開啟商品頁
- 快速結帳流程
- 會員中心查看

### 設計亮點

| 特點         | 說明                           |
| ------------ | ------------------------------ |
| **一鍵綁定** | 簡單的帳號綁定流程             |
| **分眾推送** | 按會員等級、購買歷史分群       |
| **個人化**   | 使用用戶暱稱、購買記錄客製內容 |
| **退訂機制** | 用戶可選擇退訂特定通知         |

### 技術選型

- **LINE Login**：@line/liff SDK
- **Messaging API**：line-bot-sdk
- **LINE Notify**：REST API（簡單快速）
- **Webhook**：接收用戶互動事件

### 客戶價值

- 台灣市場必備（LINE 使用率超過 90%）
- 提升訂單轉換率和回購率
- 展示第三方平台整合能力

### 預估工時

3-4 天（登入 + 通知）

## 8. PWA 離線支援

### 概述

將 Web 應用升級為 Progressive Web App，提供接近原生 App 的體驗。

### 功能特色

#### 8.1 離線瀏覽

- 產品目錄離線可查看
- 已瀏覽產品快取
- 離線時顯示友好提示
- 恢復連線後自動同步

#### 8.2 安裝到主畫面

- 「加入主畫面」提示
- 自訂 App 圖示和啟動畫面
- 獨立視窗開啟（無瀏覽器 UI）
- 支援 iOS 和 Android

#### 8.3 推播通知

- 新訂單通知
- 促銷活動通知
- 物流狀態更新
- 用戶可管理通知偏好

#### 8.4 效能優化

- 資源預快取
- 圖片延遲載入
- 背景同步

### 設計亮點

| 特點         | 說明                     |
| ------------ | ------------------------ |
| **漸進增強** | 舊瀏覽器仍可正常使用     |
| **智能快取** | 區分靜態資源和動態數據   |
| **快取策略** | 網路優先或快取優先可配置 |
| **更新提示** | 新版本可用時提示用戶刷新 |

### 技術選型

- **Service Worker**：Workbox（Google 官方工具）
- **Manifest**：next-pwa 整合
- **推播**：Web Push API

### 客戶價值

- 無需開發原生 App 即可提供類似體驗
- 提升用戶留存率
- 展示現代 Web 技術能力

### 預估工時

2-3 天

## 9. 進階搜尋系統

### 概述

升級搜尋體驗，提供更智能、更快速的產品查找功能。

### 功能特色

#### 9.1 即時搜尋建議

- 輸入時即時顯示匹配產品
- 關鍵字高亮
- 支援拼音/注音搜尋
- 最多顯示 5-8 個建議

#### 9.2 模糊搜尋

- 容錯處理（打錯字也能找到）
- 同義詞支援（「烏龍」=「青茶」）
- 中英文混合搜尋

#### 9.3 搜尋歷史

- 記錄最近 10 次搜尋
- 一鍵清除歷史
- 點擊快速搜尋

#### 9.4 熱門搜尋

- 展示當前熱門關鍵字
- 定期更新（基於搜尋頻率）
- 點擊直接搜尋

#### 9.5 進階篩選

- 價格區間滑桿
- 多分類複選
- 評分篩選
- 庫存狀態（有貨/預購）
- 篩選條件可儲存

### 設計亮點

| 特點         | 說明                     |
| ------------ | ------------------------ |
| **防抖優化** | 避免每次按鍵都發請求     |
| **鍵盤導航** | 上下鍵選擇、Enter 確認   |
| **行動優化** | 手機版全屏搜尋體驗       |
| **搜尋分析** | 後台可查看搜尋關鍵字統計 |

### 技術選型

- **基礎**：資料庫 LIKE 查詢 + 前端優化
- **進階**：Elasticsearch 或 Meilisearch
- **模糊搜尋**：Fuse.js（前端）或後端全文搜尋

### 客戶價值

- 提升用戶找到商品的效率
- 減少跳出率
- 蒐集用戶搜尋意圖數據

### 預估工時

2-3 天（基礎版）

## 10. 批量操作系統

### 概述

提供高效的批量數據處理功能，提升管理員工作效率。

### 功能特色

#### 10.1 批量導入產品

- Excel 模板下載
- 拖曳上傳 Excel 檔案
- 預覽導入數據
- 錯誤行高亮顯示
- 部分導入（跳過錯誤行）

#### 10.2 批量更新價格/庫存

- 選擇多個產品
- 統一調整（加減百分比或固定值）
- 預覽變更結果
- 確認後批量更新

#### 10.3 批量導出訂單

- 選擇日期範圍
- 選擇導出欄位
- Excel 格式下載
- 支援篩選後導出

#### 10.4 批量發送通知

- 選擇目標會員群組
- 編輯通知內容
- 預覽發送對象
- 排程發送

### 設計亮點

| 特點         | 說明                     |
| ------------ | ------------------------ |
| **進度顯示** | 大量數據處理時顯示進度   |
| **錯誤報告** | 詳細列出失敗項目和原因   |
| **操作日誌** | 記錄批量操作歷史         |
| **權限控制** | 只有管理員可執行批量操作 |

### 技術選型

- **Excel 解析**：xlsx 或 exceljs
- **背景處理**：大量數據使用 Worker 或佇列
- **進度追蹤**：Server-Sent Events 或輪詢

### 客戶價值

- 大幅提升管理效率
- 減少人工操作錯誤
- 企業客戶必備功能

### 預估工時

3-4 天
