# 完整檢查

請依序執行以下完整檢查：

## 執行步驟（依序）

### 1. 型別檢查

```bash
pnpm type-check
```

### 2. ESLint 檢查

```bash
pnpm lint
```

### 3. 完整建置

```bash
pnpm build
```

### 4. 測試（如有）

```bash
pnpm test 2>/dev/null || echo "無測試或測試略過"
```

## 錯誤處理

如有錯誤：
1. 修正錯誤
2. 再次執行失敗的步驟
3. 確認全部通過

| 建置 | [通過/失敗] |
