# 快速驗證

執行以下命令確認變更無錯誤：

## 執行步驟

### 1. 型別檢查

```bash
pnpm type-check
```

### 2. 完整建置（如果型別檢查通過）

```bash
pnpm build
```

## 錯誤處理

如果有錯誤：
1. 修正錯誤
2. 再次執行 `/verify`
3. 確認無錯誤後才進行下一步

## 輸出格式

```
# 驗證結果

## 型別檢查
- [通過/失敗]
- 錯誤數量：[N]

## 建置結果
- [通過/失敗]
- 錯誤詳情：[...]

## 總結
- 是否可提交：[是/否]
- 需修正：[列表]
```
